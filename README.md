# Ellora Sanat Evi

Ellora Sanat Evi resmi web sitesi projesi.

## Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm start
```

## Geliştirme

Bu proje React kullanılarak geliştirilmiştir.
